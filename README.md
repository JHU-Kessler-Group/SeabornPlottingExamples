#Examples of Plots Using Seaborn

This is a compilation of examples of plots using Seaborn, a powerful Python data visualization library built on matplotlib. All of the plots are taken from the example folder on seaborn's github page: https://github.com/mwaskom/seaborn/tree/master/examples

Online documentation is available at https://seaborn.pydata.org/. An example gallery is available at https://seaborn.pydata.org/examples/index.html
